#  Insurance Agency License Analysis & Risk Monitoring 

## Overview

This project analyzes insurance agency licensing data to identify compliance risks and business insights using Excel.

---

## Objectives

* Clean and standardize raw data
* Handle inconsistent date formats
* Identify active vs expired licenses
* Build dashboard-ready dataset

---

## Data Cleaning

* Removed duplicates
* Standardized column names
* Fixed mixed date formats
* Cleaned postal codes
* Handled missing values

---

## 🔧 Key Features

* License Status (Active/Expired)
* License Duration
* Clean structured dataset

---

## 📊 Analysis

* Agencies by State
* Active vs Expired Licenses
* Organization Type Distribution
* License Type Distribution
* Expired Licenses by State


---

## 🚀 Tools Used**

* Excel
* Power Query
